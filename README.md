# PAMix - the one and only pulseaudio cli mixer

# Features #
	- change sinks
	- mute sinkinputs
	- change volume

# Keybindings #
decrease volume: h
increase volume: l
down:            j
up:              k
un/mute:         m
next/prev sink:  s/S

# Planned Features #
	- modify multiple channels

# Dependencies: #
	- PulseAudio (duh)
	- Ncurses
